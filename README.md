# gas-account-book

## ハンズオン
https://qiita.com/matsu7089/items/6ea7931e97ae703f6607

## 開発環境

* Node.js v12.16.3
* yarn v1.22.4

## 使用した技術

* バックエンド
 * Google Apps Script
* フロントエンド
 * Vue.js / Vue Router / Vuex
 * Vuetify
 * axios

## インストール

```bash
git clone https://github.com/matsu7089/gas-account-book.git
cd gas-account-book
yarn install
```

## 使い方

```bash
# 開発
yarn serve

# ビルド
yarn build

# lint
yarn lint
```
