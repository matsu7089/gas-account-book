https://zenn.dev/matsu7089/books/gas-account-book

## 開発環境

* Node.js v12.18.3
* yarn v1.22.5

## 使用した技術

* バックエンド
  * Google Apps Script
* フロントエンド
  * Vue.js / Vue Router / Vuex
  * Vuetify
  * axios

## インストール

```bash
git clone https://github.com/matsu7089/gas-account-book.git
cd gas-account-book
yarn install
```

## 使い方

```bash
# 開発
yarn serve

# ビルド
yarn build

# lint
yarn lint
```
